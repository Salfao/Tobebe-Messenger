# Tobebe Messenger

**Tobebe Messenger** — безопасный веб-мессенджер от 15-летнего разработчика.

Вы сами решаете, где общаться: в обычных или **секретных чатах с E2EE‑шифрованием**.  
Создавайте каналы и группы, ведите **заметки в избранном** (включая полностью зашифрованные).  
Богатые настройки профиля для самовыражения и многое другое.

## Возможности

### Сквозное шифрование (E2EE) 

В **секретных чатах** используется настоящее сквозное шифрование:
- **AES-GCM** с 256-битным ключом
- **ECDH** для согласования ключей (P-256)
- **Perfect Forward Secrecy** — ключи меняются каждые 12 часов
- **Safety Numbers** — можно сверить ключи с собеседником вне чата

Даже я, как разработчик, не могу прочитать ваши секретные сообщения.

### Другие возможности

- Секретные чаты (E2EE, PFS, Safety Numbers)
- Группы и каналы с комментариями
- Глобальный плеер голосовых сообщений (смена скорости, визуализация)
- Избранное с тегами и **секретными заметками**
- Кастомные обои чата (кроме iOS)
- PWA — можно установить на телефон
- Аудио‑ и видеозвонки (WebRTC)
- И многое другое…

## Запуск (клиент)

1. Скачайте файлы
2. Запустите локальный сервер:  
   `npx serve .` или `python -m http.server 8080`
3. Откройте `http://localhost:3000`

Клиент по умолчанию использует API: `https://tobebe.online/api.php`
