# Tobebe-Messenger
Tobebe Messenger - безопасный веб-мессенджер от 15-летнего разработчика. Вы сами решаете, где общаться: в обычных или секретных чатах с E2EE‑шифрованием. Создавайте каналы и группы, ведите заметки в избранном, включая полностью зашифрованные. Богатые настройки профиля для самовыражения и многое другое.
